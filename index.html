Justify Content

O justify-content nos ajuda a posicionar os elementos horizontalmente, tendo vários tipos de espaçamentos diferentes.

Flex Start

O flex-start é o valor padrão do justify-content. Ele posiciona os elementos na esquerda da tela.

Flex End

Funciona de forma inversa ao flex-start. No flex-end os elementos são posicionados no fim da página.

Center

No center os elementos são posicionados no centro do container.

3

Space Around

O space-around adiciona um espaço ao redor dos elementos, mas o espaço entre os elementos fica dobrado em relação a borda, porque, como cada elemento tem o mesmo espaço, o espaço entre os elementos é somado.

1
2
3

Space Between

Basicamente funciona da mesma forma que o space-around, mas no space-between, o espaço é adicionado somente entre os elementos, não sendo adicionado espaço entre o elemento e a borda.

1
2
3

Space Evenly

O valor space-evenly também adiciona um espaço entre os elementos e a borda, mas no space-evenly os espaços são adicionados por igual em todos os lado.
